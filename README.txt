This repository holds content pertaining to Coursera's Machine Learning Course by Andrew NG.

There are text files organized by Weeks and each text file should be opened in Sublime Text. Additionally, 'Unicode Math' a Sublime plugin is needed to view the full content.
Plugin is available on github on https://github.com/mvoidex/UnicodeMath.

It can be downloaded by Tools -> Command Palette
type 'install package' and then search for 'Unicode Math' in the pop up.
This should install the plugin on Sublime.
